# 18650 Battery Shield

## 개요

18650 리튬이온 배터리 1개를 장착하여 USB 충전 및 3V/5V 전원 출력을 제공하는 보드.
본 프로젝트에서는 **모터 전원 전용**으로 사용한다. (micro:bit는 전용 배터리 팩으로 별도 급전)

## 사양

| 항목 | 값 |
|------|------|
| 배터리 | 18650 리튬이온 x 1 |
| 충전 방식 | USB (Micro-USB 또는 USB-C) |
| 출력 전압 | 3V, 5V |
| 출력 포트 | 3V+GND x 4, 5V+GND x 4 (솔더 패드) |
| 솔더 패드 총 수 | 8쌍 (16개 홀) |

## 출력 포트 배치

```
┌──────────────────────────────────────┐
│         18650 Battery Shield          │
│                                       │
│  ┌─────────────────────────────────┐ │
│  │        18650 배터리 슬롯         │ │
│  └─────────────────────────────────┘ │
│                                       │
│  [USB 충전 포트]                      │
│                                       │
│  3V 출력 (솔더 패드)                  │
│  ① 3V + GND  ── MPU6050 전원        │
│  ② 3V + GND  ── (여유)              │
│  ③ 3V + GND  ── (여유)              │
│  ④ 3V + GND  ── (여유)              │
│                                       │
│  5V 출력 (솔더 패드)                  │
│  ① 5V + GND  ── KE0136 VM (모터)    │
│  ② 5V + GND  ── (여유)              │
│  ③ 5V + GND  ── (여유)              │
│  ④ 5V + GND  ── (여유)              │
└──────────────────────────────────────┘
```

## 본 프로젝트 전원 구성 (전원 분리)

micro:bit와 모터 전원을 분리하여 모터 노이즈/전압 강하로부터 제어부를 보호한다.

```
┌─────────────────────────────────────────────────────────┐
│                      전원 분리 구성                       │
│                                                          │
│  [micro:bit 전원] ─── 전용 배터리 팩 (AAA x2, 3V)       │
│  │                    JST 커넥터로 micro:bit에 직접 연결  │
│  │                                                       │
│  └──→ micro:bit V2 ──→ I2C 3.3V ──→ MPU6050 VCC        │
│                                                          │
│  [모터 전원] ───── 18650 Battery Shield (5V)             │
│  │                                                       │
│  └──→ KE0136 VM/GND ──→ TB6612FNG ──→ DC 모터 x2       │
│                                                          │
└─────────────────────────────────────────────────────────┘

  micro:bit 전용 배터리 팩         18650 Battery Shield
  ┌──────────────────┐            ┌───────────────────┐
  │ AAA x 2 (3V)     │            │                   │
  │ JST ─→ micro:bit │            │  5V ①──→ KE0136 VM│
  │                   │            │  GND ①──→ KE0136 G│
  │ micro:bit 3.3V    │            │                   │
  │  └─→ MPU6050 VCC │            │  (3V ①②③④ 여유)  │
  └──────────────────┘            │  (5V ②③④ 여유)    │
                                   └───────────────────┘
```

### 전원 분리 장점

- 모터 구동 시 전압 강하/노이즈가 micro:bit에 영향 없음
- MPU6050 I2C 통신 안정성 향상
- 모터 전원과 독립적으로 micro:bit 동작 가능

## 주의사항

- KE0136 보드의 입력 전압 범위는 DC 6~12V이나, 5V에서도 모터 구동 가능 (토크 감소 가능성 있음)
- 모터 전류가 높으면 Battery Shield의 5V 출력 용량을 초과할 수 있음 — 모터 동작 시 전압 강하 모니터링 필요
- 충전 중에는 출력 전압이 불안정할 수 있으므로 충전과 동시 사용은 비권장
- 18650 배터리 정격 용량에 따라 동작 시간이 달라짐 (일반적으로 2000~3500mAh)
