# MPU6050 (GY-521) 6축 IMU 센서

## 개요

MPU6050은 InvenSense(현 TDK)의 6축 관성 측정 장치(IMU)로,
3축 가속도계와 3축 자이로스코프를 하나의 칩에 통합한 센서이다.

## 사양

| 항목 | 값 |
|------|------|
| 통신 | I2C (최대 400kHz) |
| I2C 주소 | 0x68 (AD0=LOW), 0x69 (AD0=HIGH) |
| 가속도 범위 | ±2g, ±4g, ±8g, ±16g |
| 자이로 범위 | ±250, ±500, ±1000, ±2000 °/s |
| ADC 해상도 | 16비트 |
| 동작 전압 | 2.375V ~ 3.46V |
| 동작 온도 | -40°C ~ +85°C |

## 본 프로젝트 설정

| 설정 | 값 | 감도 |
|------|------|------|
| 가속도 범위 | ±2g (AFS_SEL=0) | 16384 LSB/g |
| 자이로 범위 | ±250°/s (FS_SEL=0) | 131 LSB/(°/s) |
| 샘플레이트 | 500Hz (SMPLRT_DIV=1) | |
| DLPF | ~44Hz 대역폭 (CONFIG=3) | 노이즈 필터링 |

## 레지스터 맵 (사용하는 것만)

| 주소 | 이름 | 설명 |
|------|------|------|
| 0x19 | SMPLRT_DIV | 샘플레이트 분주기 |
| 0x1A | CONFIG | DLPF 설정 |
| 0x1B | GYRO_CONFIG | 자이로 범위 설정 |
| 0x1C | ACCEL_CONFIG | 가속도 범위 설정 |
| 0x3B | ACCEL_XOUT_H | 가속도 데이터 시작 (14바이트 burst) |
| 0x43 | GYRO_XOUT_H | 자이로 데이터 시작 |
| 0x6B | PWR_MGMT_1 | 전원 관리 (sleep 해제) |
| 0x75 | WHO_AM_I | 장치 ID (0x68) |

## 연결 (KE0136 보드 기준)

```
MPU6050 (GY-521)     KE0136 3PIN 헤더
─────────────────    ──────────────────
VCC  ─────────────── VCC (3.3V)
GND  ─────────────── GND
SCL  ─────────────── P19 (Signal)
SDA  ─────────────── P20 (Signal)
```

- AD0 핀: GND에 연결 (주소 0x68)
- INT 핀: 미사용 (폴링 방식)

## 상보 필터 (Complementary Filter)

가속도계와 자이로스코프 각각의 장단점을 보완하는 필터:

| 센서 | 장점 | 단점 |
|------|------|------|
| 가속도계 | 장기 안정 (드리프트 없음) | 노이즈, 진동에 민감 |
| 자이로스코프 | 단기 정밀, 부드러움 | 장기 드리프트 발생 |

**필터 공식:**
```
angle = alpha * (angle + gyro_rate * dt) + (1 - alpha) * accel_angle
```

- `alpha = 0.98` (자이로 신뢰도 98%, 가속도 2%)
- `dt`: 샘플링 간격 (초)
- 200Hz 루프 → dt = 0.005s

## 캘리브레이션

센서 초기화 시 로봇을 수직으로 세운 상태에서 200회 샘플링하여 오프셋 계산:
- 자이로: 정지 시 0이 되어야 하므로 평균값을 오프셋으로 사용
- 가속도: X/Y는 0, Z는 1g(= 수직)이 되어야 하므로 이를 기준으로 보정
