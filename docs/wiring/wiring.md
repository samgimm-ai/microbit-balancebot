# 배선도 (Wiring Diagram)

## 전체 구성도

```
  ┌─────────────────────────────────────────────────┐
  │              Keyes KE0136 확장 보드               │
  │                                                   │
  │  ┌─────────────────────────┐  ┌──────────────┐  │
  │  │    micro:bit V2 슬롯     │  │  TB6612FNG   │  │
  │  │    (골드핑거 삽입)        │  │  모터 드라이버 │  │
  │  └─────────────────────────┘  └──────────────┘  │
  │                                                   │
  │  [좌측 3PIN]              [우측 3PIN]             │
  │  P1~P10                   P11~P20                │
  │  (VCC/GND/Sig)            (VCC/GND/Sig)          │
  │                                                   │
  │  [모터 단자]                                      │
  │  V  M  A1  A2  B1  B2  G                         │
  │                                                   │
  │  [전원] DC 6-12V          [레귤레이터] 3.3V/5V   │
  └─────────────────────────────────────────────────┘
```

## 핀 매핑

### MPU6050 (GY-521) → micro:bit I2C

| MPU6050 핀 | micro:bit 핀 | 설명 |
|-----------|-------------|------|
| SDA | Pin 20 | I2C 데이터 |
| SCL | Pin 19 | I2C 클럭 |
| VCC | 3.3V | 전원 (3PIN 인터페이스 사용) |
| GND | GND | 접지 |

- I2C 주소: `0x68`
- KE0136 우측 3PIN 인터페이스 P19/P20 사용하여 연결

### Motor 1 (좌측) → TB6612FNG Channel A

| 기능 | micro:bit 핀 | TB6612FNG | 설명 |
|------|-------------|-----------|------|
| 방향 1 | P13 | AIN1 | HIGH=정방향 |
| 방향 2 | P12 | AIN2 | HIGH=역방향 |
| 속도 | P1 | PWMA | PWM 0-1023 |

### Motor 2 (우측) → TB6612FNG Channel B

| 기능 | micro:bit 핀 | TB6612FNG | 설명 |
|------|-------------|-----------|------|
| 방향 1 | P15 | BIN1 | HIGH=정방향 |
| 방향 2 | P16 | BIN2 | HIGH=역방향 |
| 속도 | P2 | PWMB | PWM 0-1023 |

## 모터 제어 진리표 (TB6612FNG)

| IN1 | IN2 | PWM | 동작 |
|-----|-----|-----|------|
| 1 | 0 | duty | 정회전 (속도=duty) |
| 0 | 1 | duty | 역회전 (속도=duty) |
| 1 | 1 | - | 브레이크 |
| 0 | 0 | - | 정지 (프리런) |

## 물리적 연결 순서

1. **micro:bit** → KE0136 보드 슬롯에 삽입 (LED 면이 바깥쪽)
2. **MPU6050** → KE0136 우측 3PIN 헤더 (P19=SCL, P20=SDA)에 연결
3. **Motor A** (좌, 1:220 기어비) → A1/A2 모터 단자에 연결
4. **Motor B** (우, 1:220 기어비) → B1/B2 모터 단자에 연결
5. **micro:bit 전용 배터리 팩** (AAA x2) → micro:bit JST 커넥터에 연결
6. **18650 Battery Shield** → 5V 출력을 KE0136 외부 전원 입력(VM/GND)에 연결

## 주의사항

- 전원 연결 전 배선 확인 (극성 주의)
- **전원 분리**: micro:bit는 전용 배터리 팩, 모터는 Battery Shield로 각각 독립 급전
- MPU6050 전원은 micro:bit I2C 3.3V에서 공급 (KE0136 3PIN 헤더 VCC)
- MPU6050은 로봇 중심, 수평으로 장착
- 모터 회전 방향이 반대이면 모터 단자 배선을 바꾸거나 코드에서 방향 반전
- I2C 핀(P19/P20)은 MPU6050 전용으로 사용 (다른 I2C 장치 공유 가능)
