## 사용 핀 요약

| 핀 | 할당 | 방향 | 용도 |
|----|------|------|------|
| **P1** | Motor L PWM | OUTPUT | 좌측 모터 속도 (PWM 0-1023) |
| **P2** | Motor R PWM | OUTPUT | 우측 모터 속도 (PWM 0-1023) |
| **P5** | Button A | INPUT | PID 파라미터 선택 (내장) |
| **P11** | Button B | INPUT | PID 값 증가 (내장) |
| **P12** | Motor L IN2 | OUTPUT | 좌측 모터 방향 2 (AIN2) |
| **P13** | Motor L IN1 | OUTPUT | 좌측 모터 방향 1 (AIN1) |
| **P15** | Motor R IN1 | OUTPUT | 우측 모터 방향 1 (BIN1) |
| **P16** | Motor R IN2 | OUTPUT | 우측 모터 방향 2 (BIN2) |
| **P19** | I2C SCL | I2C | MPU6050 클럭 |
| **P20** | I2C SDA | I2C | MPU6050 데이터 |

## 내장 기능 사용 핀 (micro:bit V2)

| 핀 | 내장 기능 | 본 프로젝트 영향 |
|----|----------|-----------------|
| P3, P4, P6, P7, P9, P10 | 5x5 LED 매트릭스 | 기울기/상태 표시에 사용 |
| P5 | Button A | PID 튜닝 파라미터 선택 |
| P11 | Button B | PID 튜닝 값 증가 |
| P19 | I2C SCL (내장) | MPU6050 연결 |
| P20 | I2C SDA (내장) | MPU6050 연결 |

## 여유 핀 (확장 가능)

| 핀 | 상태 | 확장 예시 |
|----|------|----------|
| **P0** | 미사용 | 초음파 센서 Trig/Echo, 부저 |
| **P8** | 미사용 | 추가 센서 신호 |
| **P14** | 미사용 | 서보 모터, IR 센서 |

## 연결 상세도

### MPU6050 (I2C)

```
MPU6050 (GY-521)          KE0136 우측 3PIN 헤더
┌──────────┐              ┌─────────────────┐
│ VCC ─────┼──────────────┤ VCC (3.3V)      │ P20 헤더
│ GND ─────┼──────────────┤ GND             │
│ SCL ─────┼──────────────┤ Signal (P19)    │ P19 헤더
│ SDA ─────┼──────────────┤ Signal (P20)    │ P20 헤더
│ INT      │ (미사용)      │                 │
│ AD0 → GND│ (주소 0x68)  │                 │
└──────────┘              └─────────────────┘
```

## 모터 제어 진리표

| IN1 | IN2 | PWM | 동작 | 설명 |
|-----|-----|-----|------|------|
| 1 | 0 | 0~1023 | 정회전 | 전진 방향 |
| 0 | 1 | 0~1023 | 역회전 | 후진 방향 |
| 0 | 0 | - | 정지 | 프리런 (관성) |
| 1 | 1 | - | 브레이크 | 능동 제동 |
